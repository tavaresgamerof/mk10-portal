# MK10TV - Portal Esportivo

Portal de esporte de base com transmissoes ao vivo, noticias, galeria e sistema VIP.

## Stack

- Next.js 16 + Turbopack
- React 19
- Tailwind CSS
- TypeScript
- Lucide Icons
- Framer Motion

## Como rodar

```bash
npm install
npm run dev
```

Acesse http://localhost:3000

## Admin

Acesse /admin para o painel administrativo.

## Deploy

```bash
npm run build
```

O site pode ser hospedado na Vercel, Netlify ou qualquer hospedagem que suporte Next.js.
